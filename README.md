# Samuel Amanya's Personal Website

This is a static website hosted on GitHub Pages for Samuel Amanya, an impact evaluation and analytics professional.

## Local Development

To preview the site locally:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

This will open the site in your browser at `http://localhost:8080` with live reload enabled.

## Deployment

The site is automatically deployed via GitHub Pages when changes are pushed to the main branch.

## Technologies

- HTML5
- CSS3
- No JavaScript frameworks (vanilla JS for copyright year)
